# 简介

你还在用代码生成器生成CRUD代码，你还在为加，减字段一遍遍地修改sql，还在为前后端联调写接口文档，你还在用postman一遍遍地模拟报文，你还为字典展示其含义关联字典表查询，跨域，校验、异常处理等等，这些痛点将在这里得到解决。

Boot Plus是基于spring boot构建的框架，使其开发更方便，更简单。但其不改变其原有功能，只做扩展，并且兼容原有的开发。
